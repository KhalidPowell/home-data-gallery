Histogram demo

This page exists to provide a demo for using histogram.js. First, include it in the head. Then, set up a dedicated div. Format yoru data like `data` is formatted below. Then call the `histogram` function and provide the data and the name of the container to populate.

IMPORTANT: make sure YOU set the height and width of the target container div with CSS, otherwise it might get sized to 0x0
